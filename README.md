Описание
--------

Анализ лауреатов Нобелевской премии.

Цели:
- загрузить данные о лауреатах из публичного API Нобелевского фонда
- привести данные к плоскому единому формату
- проанализировать данные

Источник данных
---------------

Источником данных является публичное API Нобелевской премии (формата JSON).

Что сделано 
----------------------

- загрузка данных и базовые проверки структуры ответа
- нормализация записей лауреатов 
- EDA
- анализ стран лауреатов (включая минимальную нормализацию данных)
- построение топ-5 стран по числу лауреатов (в целом и по отдельным категориям)
- анализ многократных лауреатов

Структура репозитория
---------------------

- Final.ipynb
  Основной ноутбук с решением, кодом и графиками.

- json_dict_processing.py
  Утилиты для извлечения значений из вложенного JSON по цепочке ключей и применения конфигов нормализации.

- prizes_configs.py, laureates_configs.py
  Конфигурация и процессор для нормализации списка призов и лауреатов.
